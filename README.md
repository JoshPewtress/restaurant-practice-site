# HTML & CSS Practice Project 1

A HTML and CSS Practice Project, a restaurant requests a simple webpage be made for them.

## Customer's Request

> I need a simple website for our restaurant. The name of the restaurant is Feast. Nothing fancy. Oh, and I need it in two days.

## Questions

| Questions | Answers |
| :-------- | :-----: |
| Will there be online ordering? | No |
| How many pages will there be? | Three, A landing page, Menu Page, About us Page |
| Will there be prices on the Menu Page? | Yes |

---

## Requirements

- Shared
    - Navigation Menu
    - Hero Image
    - Copyright Footer
- Landing Page
    - Alternating Image / Paragraph Groups
        - Crafted in the Kitchen
        - Fresh Ingredients
        - Master Chef
- Menu Page
    - Appetizers
        - Title - Price
    - Entrees
        - Title - Price
    - Desserts
        - Title - Price
- About Page
    - Alternating Image / Paragraph Groups
        - Our Story
        - Our Chef
        - Our Address
